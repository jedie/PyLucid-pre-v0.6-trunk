
PyLucid
=======

copyleft 2005 by Jens Diemer

-= DE =-
PyLucid ist ein leichtgewichtiges, OpenSource ( GPL ) Content Management System ( CMS ).
Es ist in Python CGI geschrieben. Man kann fast alle Ausgaben individuell gestalten. Es
ist kein Shell-Account nötig. Zum Betrieb ist lediglich ein Standard Webserver mit
Python (min. v2.2.1) CGI und MySQL nötig. Das Python Modul mySQLdb ist i.d.R. bei jedem
WebHoster vorinstalliert.
(PyLucid basierte auf lucidCMS)

Diese Seiten sollen einmal zum Anwenderhandbuch für PyLucid dienen.
Die aktuelle Version ist unter http://www.pylucid.org erreichbar.
Gleichzeitig bilden diese Seiten die Basis-Seiten einer frischen PyLucid-Installation.


-= EN =-
PyLucid is a lightweight, OpenSource ( GPL ) content management system ( CMS ) written
in pure Python CGI. Nearly all output can be customized. No shell account is needed. To
run PyLucid you need a standard Webserver with Python (at least v2.2.1) CGI and MySQL
( mySQLdb ). (Based on the PHP lucidCMS).

These sides are the user's handbook for PyLucid.
The current version is accessible under http://www.pylucid.org
At the same time these sides are the base-sides of of a fresh PyLucid installation.
